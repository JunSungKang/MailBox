# 사용자매뉴얼

## 웹메일 접속

브라우저에서 `http://<server>:18080`에 접속한다. 포트를 변경한 경우 `mail-server.env`의 `MAIL_WEB_PORT` 값을 기준으로 접속한다.

## 메일함

- `Inbox`: 수신 메일
- `Sent`: 웹메일에서 발송한 메일
- `Bounced`: 반송 또는 실패 메일

## 메일 작성

`Compose` 화면에서 수신자, 제목, 본문을 입력하고 발송한다. 기본 발신 주소는 `MAIL_FROM` 환경변수 값을 사용한다.

## IMAP 접속

기본 IMAP 접속 정보는 다음과 같다.

- 서버: 설치 서버 주소
- 포트: `143`
- 사용자: `mailbox`
- 비밀번호: `@@`
