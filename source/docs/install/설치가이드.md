# 설치가이드

## 준비

- Linux 서버에 Docker Engine과 Docker Compose plugin을 설치한다.
- DNS `MX`가 서버의 공인 IP 또는 메일 호스트명을 가리키게 설정한다.
- TCP `25`, `143`, `18080` 포트를 방화벽에서 허용한다.
- 운영 발신까지 사용할 경우 `SPF`, `DKIM`, `DMARC`, `PTR/rDNS`를 별도로 준비한다.

## 설치

```bash
cp mail-server.env.example mail-server.env
vi mail-server.env
```

이미지 registry 인증이 필요한 경우 로그인한다.

```bash
docker login ghcr.io
```

컨테이너 이미지를 받고 실행한다.

```bash
mail-server/compose.sh pull
mail-server/compose.sh up -d
```

## 점검

```bash
checks/preflight.sh
checks/healthcheck.sh
```

웹메일은 기본값 기준 `http://<server>:18080`에서 확인한다.
